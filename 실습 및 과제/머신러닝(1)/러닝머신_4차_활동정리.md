# HUFSAbility 비교과 프로그램 추천시스템 

이번주는 앞으로 진행할 추천 시스템 심화 프로젝트의 구체적인 계획을 세우기 위하여 아이디어 회의및 시나리오 작성을 진행하였다. 이에 기반하여 추천시스템을 직접 구현함으로서 우리만의 추천시스템을 제작하는 것을 기대한다.

아이디어 회의 방식은 다음과 같다.

1.  비교과 프로그램으로 주제를 정함

2.  각자 어떤 식으로 구현하면 좋을지, 어떤 데이터가 필요한지 등 정리

3.  각자 발표 후 아이디어 정리

4. 시나리오 작성

# 1. 추천 방법

## 1.1. 기존에 들었던 비교과 프로그램 기반 추천

사용자가 기존에 이미 들었던 비교과 프로그램을 파악한 후, 그것을 기반으로 유사한 비교과 프로그램을 추천해준다.

## 1.2. 관심있는 키워드 기반 추천

초기에 사용자가 관심 있는 키워드를 입력 또는 선택을 진행하면, 그 키워드에 해당하는, 또는 그와 유사한 키워드를 가지고 있는 비교과 프로그램을 추천해준다.


## 1.3. 나의 개인정보 기반 추천

사용자가 몇 학년인지, 어떤 학과인지, 이중 전공은 무엇인지 등 사용자가 가지고 있는 개인정보를 기반으로 사용자와 비슷한 특성을 가지고 있는 사용자들이 선택한 비교과 프로그램을 추천한다. 

<br>

# 2. 필요 데이터

## 2.1. 공통으로 필요한 데이터

 - 비교과 프로그램의 정보(신청 대상, 신청 조건, 프로그램 설명, 프로그램 리스트 등)

 비교과 프로그램의 신청 대상이 모두 다르기 때문에 고려해야 하는 부분이고, 기본적으로 어떤 프로그램인지 정보가 필요하다.

## 2.2. 기존에 들었던 비교과 프로그램 기반에서의 필요 데이터

 - 사용자가 들었던 비교과 프로그램 목록

 기존에 먼저 들었던 비교과 프로그램과 유사한 프로그램을 추천하기 때문에 그 사용자가 어떤 비교과 프로그램을 들었는지 데이터가 필요하다. 

## 2.3. 관심있는 키워드 기반에서의 필요 데이터

 - 각 프로그램이 어떤 분야에 속해있는지 
 - 키워드 목록

키워드를 중심으로 비교과 프로그램을 추천하기 때문에 각 프로그램이 어떠한 키워드에 속하는지 사전에 파악할 필요가 있고, 그 키워드에는 어떤 것이 있는지 정보가 필요하다. 

## 2.4. 사용자 개인정보 기반에서의 필요 데이터
 - 사용자의 개인정보(학과, 이중 전공, 학년, 비교과 핵심역량점수 등)
 - 누가 어떤 비교과 프로그램을 들었는지 

사용자의 개인정보를 기반으로 추천하기 때문에 사용자가 어떠한 특성을 가지고 있는지 파악하기 위해 각 사용자의 개인정보가 필요하고, 유사한 사용자가 선택한 비교과 프로그램이 무엇인지 사전에 파악해야 한다.

- 이름 : 김길동
- 학과 : GBT학과
- 학년 : 3학년
- 관심 있는 키워드 : 금융, 데이터, 클라우드, 글로벌
- 수강한 비교과 목록 :
    - ‘14차 온라인 학습법 특강<데이터를 디자인하다 : 태블로>’
    - ‘하계 방학 국내 현장 실습 사전 설명회(지원 전략 특강)’
    - ‘[데이터 센터] 데이터 전문가 육성 입문 강좌: 온.오프라인 연계 과정’
    - ‘MBTI를 활용한 진로 탐색 프로그램’
    - ‘글로벌 외국계 기업 취업 전략 특강-릴레이 8’
    - 전지적 선배 시점 Ep.05 금융 감독원편

위와 같은 정보를 가진 사용자가 추천 시스템을 이용한다고 했을 경우의 시나리오

## 3.1. 사용자 정보

기존 연구에서는 데이터가 [사용자, 아이템, 점수] 형태로 되어있어 자신과 비슷한 평점을 준 다른 사용자 중 상위 K명을 선택한 후 각 사용자들이 부여한 평점의 평균과 자신의 평점의 평균을 더하는 것으로 최종 예측을 진행. 그러나 이는 콜드 스타트 문제가 있으며, 더욱이 신입생의 경우 이 문제가 필연적으로 발생하게 됨 → 이를 해결하기 위해 기존의 데이터를 이용하기로 함.

## 3.2. 추천 목록

1. 내가 들었던 비교과 프로그램 기반으로 유사한 프로그램 추천
    - 김길동 학생이 수강한 비교과 목록과 비슷한 비교과들을 추천한다.
    - 예시
        - 2020 겨울방학 금융권 취업 아카데미
        - 진로취업지원센터(서울)
        2023 하반기 금융권 채용대비반

2. 내가 관심 있는 키워드 기반으로 그 키워드와 비슷한 프로그램 추천
    - 김길동 학생이 관심 있는 키워드를 기반으로 그 키워드와 관련된 프로그램 추천
    - 예시
        - 아마존 웹서비스(AWS) 클라우드 컨설턴트 영업 직무 선배가 들려주는 취뽀담
        - 진로취업지원센터(글로벌)
        이공계 현직자 직무 특강: IT개발(데이터) / RSQUARE Backend 개발
        - [데이터센터][데이터리안] 데이터사이언티스트 특강: 노션을 이용한 포트폴리오 작성
3. 나의 개인정보를 기반으로 비슷한 유저들이 선택한 프로그램 추천
    - 김길동 학생의 학년, 학과 등 개인정보를 기반으로 그와 비슷한 유저들이 선택한 프로그램을 추천한다.
    - 예시
        - [전략기획-캐스티파이] 글로벌기업 현직자 직무특강
        - 현직 대기업 인사담당 선배가 알려주는 "취업의 바이블”
